# Assert Flag or Scalar Integerish

Assertion function for
[`check_flag_or_scalar_integerish()`](https://smartiing.github.io/seekr/reference/check_flag_or_scalar_integerish.md).
Will throw an error if the input is invalid.

## Usage

``` r
assert_flag_or_scalar_integerish(
  x,
  .var.name = checkmate::vname(x),
  add = NULL
)
```

## Arguments

- x:

  The object to check.
